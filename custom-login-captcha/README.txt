# Custom Login CAPTCHA Plugin

## Description

The Custom Login CAPTCHA plugin adds a simple CAPTCHA to the WordPress login page, providing an additional layer of security to prevent automated bot login attempts.

## Installation

1. Download the plugin ZIP file.
2. Upload the ZIP file to your WordPress site.
3. Activate the plugin through the WordPress admin panel.

## Features

- **Basic CAPTCHA:** Adds a basic CAPTCHA to the login form.
- **Visual Challenge:** Users are required to enter a randomly generated code for authentication.

## Usage

Once the plugin is activated, a CAPTCHA will be displayed on the WordPress login page. Users must enter the code shown in the CAPTCHA to proceed with the login.

## Frequently Asked Questions

### How do I customize the CAPTCHA style?

To customize the CAPTCHA style, you can modify the `style.css` file included with the plugin. Add your custom styles to enhance the visual appearance of the CAPTCHA.

### Is this plugin secure?

The security of the plugin depends on regular updates and adherence to best coding practices. Ensure that you keep the plugin updated to the latest version to benefit from any security improvements.
